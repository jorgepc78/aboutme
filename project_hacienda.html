Mobile Apps

Patent dashboard for the Ministry of Finance of Quintana Roo, México

This app was designed for tablets and it's a dashboard showing several finances statistical indicators. This dashboard integrate maps and graphics offline. This app was focused for the principal director and managers.

  • Framework / language in Back-end : PHP
  • Framework / language in Mobile : Jquery, Bootstrap, apache Cordova
  • Mobile Platform: Android and IOS
  • Database : MySQL
  • Development year : 2014
  • Development time : 8 month
